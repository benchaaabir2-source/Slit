# Orbital Slit Simulation

Simulation classique d’un point en orbite passant par un passage étroit (slit)
et impactant un écran détecteur.

Objectif :
- R fixé
- Deux cas comparés : D < R et D > R
- Balayage du centre de gravité
- Accumulation des impacts
- Comparaison précise des distributions (cloche gaussienne, KDE, statistiques)
